一个"裸模型"和一个"被 harness 驾驭的模型"，干同一件事，差距巨大。

RAW MODEL · 裸模型

一次问答

  • 没有记忆，每次对话都从零开始
  • 不能调用任何外部工具
  • 出错没有重试机制
  • 无法跟踪、评估、优化
  • 单点风险，无安全围栏
HARNESSED AGENT · 被驾驭的 Agent

一个能干活的人

  • 拥有短期工作记忆 + 长期知识库
  • 能调用工具：读文件、写代码、查 API
  • 出错自动重试 + 人机交接条件
  • 全链路可观测 + 评测体系
  • 权限边界 + 输入输出审查

这道沟壑里的工程，就是 Harness Engineering。 Anthropic 2026 OpenAI 2026 先后定义了这个领域。

WHAT · 是什么

一句话
讲清楚

Agent = Model + Harness。模型负责"想"，harness 负责让它"能干活"。

Agent = Model + Harness

Harness（驾驭套件）= 包在模型外面、把被动模型变成能干活 Agent 的那一整圈代码与结构。

它有三个尺度 —— 组件级（裹一次模型调用）⊂ 任务级（编排多 Agent 干成一件有终点的事）⊂ 组织级（罩一支 Agent 队伍 + 一个长期产品）。

三个尺度的定义不冲突，只是站在"尺度尺"的不同刻度上。 LangChain Martin Fowler

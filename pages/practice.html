docs/ 三件套

三个 markdown 文件 + 一个决策追加目录 —— 每个都有 placeholder 结构，学员填空即可。

docs/decisions/ —— "只追加不重写"的决策考古： 每次有重大决策（改方向 / 砍 feature / 换技术栈），追加一份日期文件。 永不重写历史决策 —— 新 agent 接手项目，翻一遍就能复盘"为什么走到了今天"。 这是 OpenAI "环境比模型重要"的应用层实现。

Hook 跟 prompt 的区别

软提醒写 prompt，硬规矩写 hook。 CLAUDE.md 里的"边界"是软提醒，模型可能忘。 hooks/pre_tool_use.sh 是确定性代码 —— 由 Claude Code 在工具调用前真去跑，模型绕不开。

Hook 三个关键： 退出码 0 = 放行，≠0 = 阻止； 阻止时往 stderr 写 DENIED 信息； 脚本必须 chmod +x 可执行。

评测：两个文件 + 一个独立 Reviewer

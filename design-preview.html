怎么挑（写给思敏）

  • 🅰️ 选 A：如果目标是"让面试官 / HR / 工程师一眼觉得你专业"。最克制，最像 Linear / Vercel 的官方文档。
  • 🅱️ 选 B：如果想要"高品质 + 有温度"，像 Anthropic / Stripe 的工程博客。衬线字会显得"是一份认真的研究"。
  • 🅲 选 C：如果想要最炫酷，符合"AI 产品经理课程发布物"的传播效果。Hero 一秒抓眼球。
  • 🔀 混搭也行：例如 C 的炫酷 hero + A 的克制内页（首页爆点 + 内页可读性），或 C 的渐变 + B 的衬线标题。
